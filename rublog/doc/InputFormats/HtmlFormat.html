HTML Format

A file ending in .htm and .html is passed straight through. If it contains <title>stuff</title> or <h1>stuff</h1>, then that text is used at the title, otherwise the first line of the file is used.

It possible to include complete HTML files as RubLog articles. However, RubLog does no particular parsing of HTML files, so the resulting log, while displaying propertly in all browsers I've tried, won't validate as HTML.

Text in the file between <summary> and </summary> is an article summary: it remains in the article, and will be used as the article's synopsis if synopsis mode is enabled..
